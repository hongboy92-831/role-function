系統總覽

本區塊整理玩家成長系統的主幹與章節開放節奏，作為後續詳細規格的快速索引。

武器
三把武器可戰鬥中自由切換，分別提供均衡、快速、重型範圍三種手感。
戰技點數
所有武器共用戰技點數。戰技點數透過攻擊敵人恢復，用於施放各武器戰技。

義肢技能可即時切換，服務探索、解謎與戰鬥輔助。

圖鑑系統
記錄玩家探索時遇過的敵人，提供怪物詳細介紹，強化世界觀理解與代入感。

一、設計原則

玩家成長系統以清楚、直覺、低操作負擔為核心。武器、義肢、符文與戰技各自有明確用途，但不彼此綁定成複雜組合。

武器
可在戰鬥中自由切換，主要差異為攻擊手感、攻速、範圍、破防與戰技用途。
戰技
每把武器各自擁有戰技，消耗共用戰技點數，戰技點數透過攻擊敵人恢復。
義肢技能
可在戰鬥中即時切換，用於解謎、探索與戰鬥輔助，定位接近《死亡之門》的工具型技能。
符文
只提供被動加成，不改變戰鬥方式，不與特定武器、義肢或戰技綁定。
基礎成長
生命刻印收集 3 個後提升最大生命，戰意刻印收集 3 個後提升戰技點數上限，取得後自動累積，不需要升級 UI。
時痕碑
符文只能在時痕碑更換；時痕碑同時作為存檔點與符文配置入口。
商人
販售補救與便利型商品，包含部分符文與少量星核零件，限量販售。
換裝
男／女主角服裝更換屬於外觀功能，服裝來源包含預購遊戲、關卡解謎獲得與節慶發送。
圖鑑
玩家在探索時遇過的敵人會登錄至圖鑑，提供怪物詳細介紹與世界觀補充。
武器、符文、義肢、戰技之間採用平行支撐關係。玩家不需要因為切換武器而重新配置符文，也不需要依賴特定義肢或符文才能發動某個戰技。

二、武器與戰技

三把武器可於戰鬥中自由切換。每把武器擁有獨立戰技，但戰技皆消耗共用戰技點數。武器不與符文或義肢技能形成組合技。

武器 定位 戰技 戰技效果 設計重點
律法大劍 均衡型大劍。平砍不過度笨重，兼具穩定輸出與破防入門。 星辰審判 消耗戰技點數，凝聚星辰劍影向前重劈。可三段蓄力，蓄力越高傷害與破防越高。命中持盾、重甲或大型敵人時，可大幅削減防禦狀態。 適合作為主武器與玩家初期熟悉戰技破防的核心手段。
瞬影匕首 高速型武器。出刀速度極快，如影子閃現，適合追擊、小怪與靈活敵人。 瞬影連斬 消耗戰技點數，向前短距離突進，對路徑上的敵人造成多段斬擊。 強調速度與連擊感；破防能力較低，不取代律法大劍的破防定位。
墜星巨劍 重型範圍武器。攻擊慢、範圍大、破防高，適合群體壓制與大型敵人。 星殞迴旋 消耗大量戰技點數，短暫蓄力後，玩家與墜星巨劍一同進入迴旋狀態。持續期間可控制移動方向，對路徑上的敵人造成連續斬擊、擊退與破防累積。小型敵人若承受完整傷害，通常會被直接擊倒；對重型敵人與 Boss 則以破防與壓制為主。 提供後期範圍壓制能力。路徑上的敵人不設定為必定斬殺，避免壓過其他武器。

戰技共通規則

  • 所有戰技消耗共用戰技點數。
  • 戰技點數透過攻擊敵人恢復。
  • 每把武器的戰技只服務該武器本身，不與其他武器、義肢技能或符文產生連動條件。
  • 戰技定位需保持清楚：星辰審判負責蓄力破防；瞬影連斬負責突進多段；星殞迴旋負責範圍壓制。

三、義肢技能

義肢技能為主角左手機械義肢的工具模組，可於戰鬥中即時切換。義肢技能同時服務探索、解謎與戰鬥輔助，但不與武器戰技或符文形成複雜連動。

取得時機 義肢技能 核心定位 可用情境 限制
第一章 刻魂刃 遠距精準命中 遠距攻擊、命中遠處機關、穿過火源形成火刃、點火盆、燒菌絲、引爆孢子囊。 不作為主要武器替代品；平常狀態一次只能存在 1 發，時停狀態最多可預放 3 發。
第二章前半 鉤索 位移與追擊 對特殊鉤點施放，將玩家拉向鉤點；對指定敵人施放，將玩家拉向敵人身邊，用於閃避危險與貼身攻擊。 不是把敵人拉到玩家身邊；重型敵人與 Boss 需使用指定部位或鉤點。
第二章後半 磁引脈衝 中距離磁力干擾 啟動、停止或移動金屬機械裝置；破牆、推動重物，並破壞護盾。 不自由吸拉所有物件，不控制普通 NPC，不與戰技、武器、符文綁定。
第三章 時空殘影 位置回溯與危險迴避 義肢會在玩家數秒前的位置留下時間殘影，施放後玩家回到殘影所在位置。主要用於避開大型攻擊、脫離危險區域、修正站位與解決需要位置回溯的機關。 只回溯玩家位置，不回復生命，不重置敵人、機關、道具或場景狀態；需有明確冷卻與殘影視覺提示。

四、義肢強化系統

義肢強化於第一章結束後開啟。強化素材只使用一種「星核零件」，所有已取得義肢技能都可強化。每個義肢技能最高 3 級。

強化素材

素材 用途 取得來源 販售規則
星核零件 義肢技能通用強化素材。 小怪低機率掉落、精英敵人、關卡密室寶箱、支線獎勵、能力回收區、小 Boss、中 Boss。 商人可少量限量販售，賣完不補，作為補救手段。

五、符文系統

符文是被動強化系統。玩家只能在時痕碑更換符文，符文不改變基礎操作方式，不與武器、義肢技能或戰技形成組合技。

六、基礎成長道具

基礎成長道具用於提升玩家最大生命與戰技點數上限。此成長不透過升級功能或角色 UI 操作，而是透過探索與挑戰取得道具後自動累積。

道具 累積規則 提升效果 設計目的
生命刻印 收集 3 個後自動生效 最大生命 +1 提供永久生存成長，作為探索與支線回報。
戰意刻印 收集 3 個後自動生效 戰技點數上限 +1 提升武器戰技可使用空間，強化戰鬥歷程。
生命刻印與戰意刻印取得後只進行累積提示，集滿 3 個時自動提升對應能力。此系統不佔符文容量、不消耗星核零件，也不需要額外升級介面。

七、商人販售規格

商人提供補救與便利選項，不販售主線能力。商品以限量為主，賣完不補。

符文販售

上架時機 商品 用途
第一章結束後生息符文、輕步符文提高生存與跑圖舒適度，作為符文系統初期選項。
第二章中段疾行符文提升閃避距離，配合第二章更高頻率的位移與戰鬥。
第二章後半戰意符文玩家已理解戰技點數循環後，提供戰技資源強化選項。
第三章中段精準符文此時玩家已擁有多個義肢技能，義肢傷害加成具有明確價值。

八、換裝系統

換裝系統為外觀收集與展示系統。玩家可更換男／女主角服裝，服裝不影響武器、戰技、義肢技能、義肢強化、符文容量或角色數值。

項目 規格內容
功能定位 提供男／女主角服裝更換，作為外觀收集、預購回饋、關卡解謎獎勵與節慶發送內容。
服裝效果 僅改變男／女主角外觀呈現，不提供戰鬥數值、不改變戰鬥方式，也不影響符文、義肢或武器規則。
更換方式 玩家可於指定介面更換已取得服裝；實際入口可後續依 UI 架構規劃。

服裝來源

來源 說明
預購遊戲 可作為預購版本或早期購買回饋服裝。
關卡解謎獲得 可作為支線解謎、隱藏房間、捷徑回收或特殊挑戰的外觀獎勵。
節慶發送 可作為節慶活動、營運發送或期間限定紀念服裝。

九、圖鑑系統

圖鑑系統用於記錄玩家在探索與戰鬥中遇過的怪物。玩家可在圖鑑中查看已遇過怪物的詳細介紹，補充敵人背景、習性與世界觀資訊，提升探索後的代入感。

項目 規格內容
功能定位 作為怪物資料收藏與世界觀補充系統，概念接近《空洞騎士》的圖鑑。
登錄條件 玩家在探索或戰鬥中遇到怪物後，該怪物會登錄至圖鑑。
顯示內容 顯示已遇過怪物的名稱、外觀識別、詳細介紹、背景描述與相關世界觀資訊。
系統目的 讓玩家在探索後能回看敵人資料，理解怪物與章節環境、文明、詛咒或事件之間的關聯。

十、待設計架構內容

  • 義肢技能可同時服務探索、解謎與戰鬥輔助、開啟支線、捷徑、隱藏寶箱。

  • 星核零件可透過主線、支線、密室、精英敵人與商人取得。
  • 符文可提供被動加成與配置傾向。
